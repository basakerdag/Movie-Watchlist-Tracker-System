# Movie-Watchlist-Tracker-System

 Movie Watchlist Tracker System is a full-stack movie tracking dashboard developed as a System Analysis and Design course project. The architecture is built upon two distinct but harmonious data layers. First, a custom internal API (built with Node.js, Express, and SQLite) manages the primary local dataset, handling all CRUD operations for the user's personal catalog, including watch statuses, categories, and personal ratings. Second, to enrich this local dataset without bloating the database, the system features a secure backend proxy that integrates an external service: the OMDb API. This external API automatically fetches dynamic Hollywood metadata (such as plot, cast, and high-resolution posters) using precise 'Title + Year' filtering. To ensure data integrity across both the internal database and the external proxy, comprehensive unit and integration tests were written using Jest and Supertest. To run the project locally, execute `npm install` followed by `node server.js`, and open `http://localhost:3000` (Swagger API documentation is available at `/api-docs`).
